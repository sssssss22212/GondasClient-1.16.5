# GondasClient v2.0.0

**Minecraft 1.16.5 Forge Client Mod**

## Улучшения в версии 2.0.0

### 🚀 Новые функции
- **Поддержка мобильных лаунчеров** - автоматическое определение PojavLauncher, MCLauncher, Fold Craft Launcher

- **Новые настройки** - больше опций для каждого модуля

- Улучшена система конфигурации с автосохранением
- Добавлена проверка версии конфига
- Оптимизирован рендеринг ESP, Tracers и других визуальных модулей
- Улучшена совместимость с античитами (NCP, AAC, Matrix, Grim)
- Добавлен toggle cooldown для предотвращения случайных нажатий на мобильных

### 📱 Мобильная оптимизация
- Автоматическое определение мобильного лаунчера
- Адаптивный масштаб интерфейса
- Оптимизация рендеринга для слабых устройств

## Требования для сборки

- **Java 8 JDK** (OpenJDK 8 или Oracle JDK 8)
- **Gradle 7.x** (автоматически скачивается через wrapper)

## Инструкция по сборке

### Linux/macOS:
```bash
chmod +x build.sh
./build.sh
```

### Windows:
```cmd
gradlew.bat build
```

### Ручная сборка:
```bash
./gradlew build
```

Готовый JAR файл будет в: `build/libs/GondasClient-1.16.5-2.0.0.jar`

## Установка

### Компьютер:
1. Скопируйте JAR файл в папку `mods` вашего Minecraft
2. Убедитесь, что установлен Forge 1.16.5

### PojavLauncher (Android):
1. Скопируйте JAR в `/games/PojavLauncher/mods/`
2. Или через меню лаунчера: Настройки → Управление модами → Добавить

### MCLauncher / Fold Craft Launcher:
1. Скопируйте JAR в папку mods соответствующего профиля
2. Убедитесь, что Forge 1.16.5 установлен

## Использование

- **RIGHT SHIFT** - открыть ClickGUI
- ЛКМ по модулю - включить/выключить
- ПКМ по модулю - открыть настройки
- Колёсико мыши - прокрутка списков
- ESC - закрыть ClickGUI

## Настройка клавиш

Каждому модулю можно назначить свою клавишу через настройки Minecraft:
`Настройки → Управление → Категория: Gondas`
